888. Fair Candy Swap

給定兩個整數陣列：
A：Alice 擁有的糖果大小
B：Bob 擁有的糖果大小
Alice 和 Bob 想交換 一顆糖果，讓雙方交換後糖果總和相等。
請回傳一個長度為 2 的陣列 [x, y]：
x 是 Alice 要給出的糖果
y 是 Bob 要給出的糖果
若有多組解，只需回傳其中一組。

實際生活或工程上的應用
  一、資源交換與平衡（Resource Balancing）
  二、負載均衡（Load Balancing）
  三、交易與公平交換（Fair Trade / Barter System）
  四、教育與團隊分組（Balancing Groups）
  五、資料分配與同步（Data Synchronization）
  
| 步驟         | 意義     | 時間       |
| ---------- | ------ | -------- |
| 求總和        | 比較兩方總量 | O(n + m) |
| 求差值        | 算出需補的量 | O(1)     |
| 用 set 加速查找 | 降低查找成本 | O(1)     |
| 掃描 A       | 找對應 B  | O(n)     |
